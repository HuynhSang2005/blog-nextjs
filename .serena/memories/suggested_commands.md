# Suggested Commands (Windows / Bun)

## From repo root
- Dev (turbo): `bun dev`
- Build (turbo): `bun run build`
- Lint (turbo): `bun run lint`
- Lint+fix (turbo): `bun run lint:fix`

## apps/web
- Dev: `cd apps/web; bun dev`
- Typecheck: `cd apps/web; bun run typecheck`
- Lint/format check: `cd apps/web; bun run lint`
- Lint/format fix: `cd apps/web; bun run lint:fix`
- Seed: `cd apps/web; bun run seed`

## Shadcn UI
- Thêm component: `cd apps/web; bunx shadcn@latest add button`
- Audit nhanh sau khi thêm: `cd apps/web; bunx shadcn@latest add --help`

## Git
- Status: `git status --porcelain`
- Diff: `git diff`

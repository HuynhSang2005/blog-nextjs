## Quick Start

### Prerequisites

- **Bun** >= 1.3.5 ([Install Bun](https://bun.sh/))
- **Node.js** >= 20.0.0 (for compatibility)

### Installation

```bash
# Clone repository
git clone https://github.com/HuynhSang2005/blog-nextjs.git
cd blog-nextjs

# Install dependencies
bun install

# Start development server
bun dev
```
